Website de gerenciamento de pedidos de resgate para a ong "Crazy Cat Gang"

Website desenvolvido com o intuito de auxiliar a ong no gerenciamento de pedidos de resgates de gatos.

Problema

A problemática escolhida foi a falta de uma gestão de pedidos de resgate no site da ONG. A ausência dessa implementação dificulta ou até impede que usuários que usariam do site para ajudar no resgate de um gato consigam auxiliar, ou seja, é um cofator atenuante de um problema já enraizado.

Solucao

Para resolver o problema selecionado pela nossa equipe, surgiu-se a ideia de deixar mais visível a área de resgate de gatinhos e facilitar o acesso dos usuários que querem ajudar esses animais. A partir dessa ideia, foi criado um botão na página principal da ONG Crazy Cat Gang. Este, por sua vez, levará o usuário a uma página na qual ele preencherá um um relatório, com informações essenciais para o resgate dos gatos pela ONG. Além disso, foi criada uma página com as imagens dos gatos resgatados, a fim de gerar confiança e criar um vínculo entre ONG e solicitante.
