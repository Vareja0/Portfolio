Jokenpo python

Programa desenvolvido inteiramente em python. O jogo consiste em três modalidades: Humano x Humano, Humano x Computador, Computador x Computador. Uma vez que o usuário escolha a modalidade, será executada apenas a modalidade escolhida. Para trocar sua escolha, o usuário terá que reinicializar o programa.
